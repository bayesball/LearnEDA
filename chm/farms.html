farms(LearnEDA)R Documentation

Number of farms in the states of the U.S.

Description

The number of farms (in 1000s) for each of the 50 states in the United States in 1999.

Usage

farms

Format

A data frame with 50 observations on the following 2 variables.

state
name of the state
count
count of the number of farms

Source

2001 New York Times Almanac
