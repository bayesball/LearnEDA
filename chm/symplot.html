symplot(LearnEDA)R Documentation

Symmetry plot

Description

Constructs a symmetry plot.

Usage

symplot(d)

Arguments

d numeric vector of values

Author(s)

Jim Albert

Examples

# symmetry plot for normally distributed data
symplot(rnorm(100))
# symmetry plot for exponential data
symplot(rexp(100))
